# SQLTrain

SQLTrain is an interactive web application for learning and practicing SQL.

The project will include:

- an in-browser SQL editor
- a Northwind-style training database
- editable database tables
- CSV and Excel imports
- interactive SQL exercises
- automatic query validation
- database reset and restore functionality

## Tech Stack

- Next.js
- TypeScript
- React
- Tailwind CSS
- SQLite WebAssembly
- Monaco Editor
